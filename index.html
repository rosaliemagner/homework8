Homework 8: Vector Geospatial Analysis for Emergency Environmental Response

Research Question

This project uses vector geoprocessing analysis to answer the following real-world question: How much of New Jersey's historic shoreline is in a coastal flooding area?

Background

I downloaded shapefile data that displayed the historic shoreline in New Jersey and the coastal flooding areas from the last 100 years.


Methods and Results

I used the geoprocessing tool Dissolve to combine the coastal flooding areas in the same zones, getting rid of unnecessary boundary lines. I used the geoprocessing tool intersection to determine the areas of the historic shoreline that intersect with coastal flooding area boundaries from the last 100 years.
